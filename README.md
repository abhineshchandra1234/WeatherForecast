# WeatherForecast
- Weather Forecast App
- It will tell the weather of the locations in the US
- Users can enter a new city and also bookmark thier city
- It will show whether data for the current week

# Tech Stack
- Coroutine
- Coil
- Retrofit
- Room
- Jetpack compose
